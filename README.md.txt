# Auto Data Visualization Platform

An interactive Python application that automatically generates **all possible visualizations** for any dataset (CSV/Excel).  
Designed to help users quickly explore data without writing any code.

## ðŸš€ Features

- Upload any CSV or Excel file.
- Automatic **data type detection** (numeric, categorical, datetime).
- Select **columns** for visualization.
- Auto-generate **multiple chart types** based on column type:
  - Numeric: Histogram, Box Plot, Line Chart, Violin Plot, Scatter Plot, Correlation Heatmap
  - Categorical: Bar Chart, Pie Chart, Donut Chart
  - Datetime: Time Series with numeric column
- Interactive charts using **Plotly**.
- Built with **Streamlit** for fast and user-friendly web interface.
